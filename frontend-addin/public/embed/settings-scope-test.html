How to run the test:
  1. Insert two instances of this add-in onto two different slides.
  2. In instance 1 click "Write A". In instance 2 click "Write B".
  3. Save the .pptx. Close PowerPoint.
  4. Reopen the file. Each instance auto-reads its value on load.
  5. If instance 1 shows "A" and instance 2 shows "B" → per-instance settings confirmed. If both show the same value → shared bag confirmed.
